About this project

This small project is the first step in building a portfolio. It contains valid HTML structure, clear semantics, and helpful accessibility attributes.

Why semantics matter

Using proper headings and sectioning elements helps screen readers and search engines understand the content better.

Every paragraph here is intentionally short and focused — good practice when learning to structure content for the web.

This page also includes a responsive meta tag so the layout behaves well on phones and tablets.
